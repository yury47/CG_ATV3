uint32_t abcg::VulkanImage::getMipLevels ( ) const
noexcept

Returns the number of mipmap levels generated for this image.

If the image is created with generateMipmaps = false, the number of mipmap levels is always 1. Otherwise, it is computed as \(\lfloor \log_2(\max(w, h)) \rfloor + 1\), where \(w\) and \(h\) are the texture width and height.

Returns
Number of mipmap levels.
