Detailed Description

Base class for a window that displays graphics using an OpenGL context.

Derive from this class to create a custom OpenGL window object to be passed to abcg::Application::run.

See also
abcg::OpenGLWindow::onEvent for handling SDL events.
abcg::OpenGLWindow::onCreate for initializing OpenGL resources.
abcg::OpenGLWindow::onPaint for scene rendering.
abcg::OpenGLWindow::onPaintUI for UI rendering.
abcg::OpenGLWindow::onResize for handling of window resize events.
abcg::OpenGLWindow::onUpdate for commands to be called every frame.
abcg::OpenGLWindow::onDestroy for cleaning up OpenGL resources.
Remarks
Objects of this type cannot be copied or copy-constructed.

◆ onPaint()

void abcg::OpenGLWindow::onPaint ( )
protectedvirtual

Custom handler for rendering the OpenGL scene.

This virtual function is called for each frame of the rendering loop, just after abcg::OpenGLWindow::onPaintUI.

This is not called when the window is minimized.

Override it for custom behavior. By default, it clears the color buffer and calls glViewport(0, 0, w, h), where w is the width, and h is the height of the window.

◆ onPaintUI()

void abcg::OpenGLWindow::onPaintUI ( )
protectedvirtual

Custom handler for rendering Dear ImGUI controls.

This virtual function is called for each frame of the rendering loop, just before abcg::OpenGLWindow::onPaint.

This is not called when the window is minimized.

Override it for custom behavior. By default, it shows a FPS counter if abcg::WindowSettings::showFPS is set to true, and a toggle fullscreen button if abcg::WindowSettings::showFullscreenButton is set to true.
